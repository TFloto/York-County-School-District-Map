To start off, you may need to install node.js onto your computer. 
-If node is not installed

-double left click on node_setup.exe and go through the installation process

To run the server-

-double left click on start.bat

Start.bat actually starts the server and automatically downloads all the dependencies. 




In PyInstallers is the original node_setup.py file that was used for the .exe file. It was left there in case the .exe file needed change. In the case the .exe file needed changed-

-go to /PyInstallers
-change info in node_setup.py
- open a terminal and type in ‘pyinstaller –onefile mode_setup.py’
-once completed go to /dist and copy the .exe file to the root project folder
